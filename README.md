# OpenMod

A modern Stoat bot built with [Node.js](https://nodejs.org) and [revolt.js](https://revolt.js.org).

## Features

- Full moderation suite (kick, ban, warn, mute)
- Auto-moderation (anti-spam, bad word filter)
- Reaction roles for self-assignable roles

## Requirements

- Docker
- Docker Compose
- A Stoat bot token
- A Stoat server

## Quick Start

⚠️ You need to setup the `.env` file based on the `.env.example`.

```sh
# Start the bot
docker compose up -d
```

## Commands

All commands use `/` prefix by default.
